# uni-colorui

#### 1.[安装](https://www.color-ui.com/)

```
npm i -S uni-colorui
```

#### 2.使用 uni-app -> App.vue -> 引入全局样式

```
<style>
@import "uni-colorui/theme/main.css";
@import "uni-colorui/theme/icon.css";
</style>
```

#### 3.[github仓库](https://github.com/weilanwl/ColorUI)

